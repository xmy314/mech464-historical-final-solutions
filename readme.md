## Contributing

Spotted an error or have a better approach? Contributions are welcome.

**To fix something**: fork the repo, make your changes, and open a pull request with a brief explanation.

**To flag an error or discuss**: open an issue referencing the specific question (e.g. `y2023 Q1-1 error`).

## Disclaimer

These are **unofficial** solutions, not course staff. They may contain errors. Always think critically and verify against your own understanding.
